Tankaga Project

Home Page Projects Page

My first serious computer science project. This is a tank game created in Unity written in C#. Two tanks try to destroy each other from across a play field while simultaneously dodging shots by jumping and moving. This is a multiplayer networked game utilizing Unity's (now deprecated) UNET networking system for multiplayer games. There is an older, local multiplayer version of this game which does exist and is 2D, but it is currently unavailable on for viewing on Github.
