Home Page Projects Page

An action rpg game demo created for the purposes of learning humanoid and monster animation, modeling, texturing, UI, and state machine AI. This game is currently a work in progress. The demo features a single monster with 5-10 attack animations of different forms. The player character features 6 different attack animations as well as a run, jump, dodge, and grappling hook. The monster is immune to player attacks at first, but through interacting with the world, the player can work to defeat the monster and beat the game.
