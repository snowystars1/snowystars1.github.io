Xamarin Alphanumeric Android Application

Home Page Projects Page

This is a project I worked on during my Spring 2017 semester in highschool because I was interested in android applications. The application is fairly simple and was created with the heavy use of tutorials and guides. This app allows a user to type in an alphanumeric phone number like: 1-800-XAMARIN and then the application will turn that alphanumeric phone number into a number which is actually callable. Also the application features a history so the user can see a history of previously translated phone numbers.
